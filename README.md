# Небольшой помошник для ДнД водилы (изучаю Express, vue3, NETCore)
* Открывать в visual studio 2026
## запускать
* NETCore webAPI: `dotnet run` запускать из DND.NETCore.API
* Express webAPI: `node www` запускать из ExpressServer/bin
* vue client `npm install` `npm run dev` запускать из vueclient
* СУБД запускаем в докере: `docker-compose.yml` пока что ручками включаем образ
* в папку с Nginx кладём конфиг из корня репозитория. а находится в E:\nginx-1.27.1\nginx.exe
